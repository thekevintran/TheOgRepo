This repository is an app made for Android phones.
The title of the app is TechSpotInventory.
The app was created for the CAPSTONE (COMP-7500-C62) Course at Wentworth Institute of Technology by Kevin Tran.

Instructions:
The repository should be forked via Github to your own Github repository before taking further steps.
This code repository should be forked, pulled, and compiled within Android Studio 3.1.1 or a further updated version of Android Studio.
The local SDK should be the primary choice when loading the repository to Android Studio.

Hardware:
The Android Virtual Device chosen for testing is: Nexus 5X API 26 x86   
Utilizing Android 8.0
Storage: 3GB
